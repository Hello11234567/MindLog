# 🧠 MindLog
> 캡스톤디자인 프로젝트 | 2025.10 ~ 2025.12

Flutter 기반 AI 감정 관리 애플리케이션입니다.
청소년 및 청년들이 일상에서 자신의 감정을 기록하고,
AI 분석을 통해 감정 상태를 인식할 수 있도록 돕는 서비스입니다.

> 본 프로젝트를 기반으로 Spring Boot 백엔드 + 출시를 목표로 한
> 개인 프로젝트 [마음이(Maeum)](https://github.com/Hello11234567/Maeum) 를 개발 중입니다.

## ✨ 주요 기능
- 🎚 슬라이더 기반 5가지 감정 수치 입력 (기쁨, 화남, 불안, 평안, 슬픔)
- 🤖 OpenAI GPT API 기반 감정 분석 및 한 줄 요약
- 💡 감정 상태 기반 개인화 케어 추천
- 📊 주간/월간 감정 통계 시각화 (레이더 차트)
- 📅 캘린더 기반 감정 기록 확인 및 AI 대표 이모지 표시

## 🛠 기술 스택
| 구분 | 기술 |
|------|------|
| Frontend | Flutter, Dart |
| Auth | Firebase Authentication |
| Database | Cloud Firestore |
| AI | OpenAI GPT API |

## 👩‍💻 담당 역할 (4인 팀 / 핵심 기능 전담)
- 팀장 / PM — 프로젝트 전체 기획 및 일정 관리
- Firebase 데이터 구조 설계 및 구현
- OpenAI API 연동 및 감정 분석 기능 구현
- 앱 UI 전체 구현 (감정 입력, AI 분석 결과, 통계 화면)
- 주간/월간 감정 통계 계산 로직 설계 및 시각화
